## üìö Intro

MineSweeper is a web application built using ASP.NET Core, featuring Identity authentication and a React frontend. The application also incorporates SignalR for real-time chat updates during gameplay.

## üåê Prerequisites

Before running the application, make sure you have the following:

    - .NET SDK installed
    - SQL Server database with a connection string configured in the appsettings.json file

## ‚öôÔ∏è Installation

To install and run the application, follow the steps below:

    Clone the repository to your local machine.

    Open the solution in Visual Studio.

    Navigate to the appsettings.json file and update the SQL Server connection string to match your database configuration.

    Build and run the application in Visual Studio.

! Don't forget to [Link your Database](#database-configuration)

## üñ•Ô∏è API Documentation

### This is our API endpoint for starting a game session upon receiving a request. It handles user authentication, game creation, and returns relevant game data.

```c#
[HttpPost("startgame")]
public IActionResult StartGame([FromBody] CreateGameRequest request)
{
    var userId = User.FindFirst(ClaimTypes.NameIdentifier)?.Value;

    var game = CreateNewGame(request, userId);

    _context.SaveChanges();

    var gameView = GameConverter.ConvertGame(game);
    return CreatedAtAction(nameof(GetGame), new { id = game.Id }, gameView);
}

```

## üìë API Endpoints and Data

The SPAminesweeper API provides the following endpoints and returns the associated data:

| API Endpoint                                      | Returned Data                                                                                                                |
| ------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| POST /api/game/addscore                           | Adds a score to the database for a specific game.                                                                            |
| POST /api/game/flagtile                           | Toggles the flag status of a tile on the game board.                                                                         |
| GET /api/game/gettopfiveoverall?isDaily={isDaily} | Retrieves the top five scores, optionally filtered by daily scores.                                                          |
| GET /api/game/getuser                             | Retrieves information about the currently logged-in user, including their username, nickname, total score, and games played. |
| GET /api/game/{id}                                | Retrieves game information by specifying its ID.                                                                             |
| POST /api/game/revealtile                         | Reveals a tile on the game board based on the provided request.                                                              |
| GET /api/game/userloggedin                        | Checks if a user is currently logged in. Returns true if logged in, false otherwise.                                         |
